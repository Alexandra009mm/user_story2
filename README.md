# Inventory Management System

## Description

This project is a simple Python program that allows users to manage an inventory. The user can add products, view the inventory, and see basic statistics such as total inventory value and total number of products.

## How it works

The program displays a main menu where the user can choose different options. The user can add products by entering the product name, price, and quantity with proper validation. The inventory is stored in a list of dictionaries.

The user can also view all registered products in an organized format. In addition, the program provides a statistics menu where the user can calculate the total value of the inventory and the total number of products.
